• Pwd -> текущая дериктория
  • ls -> все файлы в директории, ls -la -> все файлы в директории + скрытые файлы
  • cd .. -> одна дериктория вверх, cd ../../
  • clear -> очистка
  • ctrl + l -> очистка
  • c:, d:, e:, и тд. -> переход на диск сразу
  • touch -> создание файлов
  • mv -> переименование, перемещение
